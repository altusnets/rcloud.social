Simon Urbanek will be presenting RCloud - Collaborative Environment for Visualization and Big Data Analytics at the use R! 2016 Conference taking place from June 27 - 30, 2016. Analyzing Big Data in real life poses challenges with respect to performance, methodology and reusability. R is well known for its succinct syntax for analytic tasks as well as plethora of tools for data analysis and visualization, but it is not always associated with scalability. In this talk we will use examples to present R tools that allow out-of-core processing of data interfacing different back-ends including Hadoop without sacrificing the ease of use of R. We will introduce RCloud, a collaborative environment that enables sharing and scalability for teams of data scientists, featuring flexible framework for interactive visualizations and easy re-use of code and analyses.
