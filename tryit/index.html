Try It

Try It    Online or Locally

There are two methods of using the public instance of RCloud; as a Guest with read only access or as an Data Scientist with developer capabilities; the following sections describe the requirements for each profile.   Alternatively, a local private instance of RCloud may be set up so that analyses are only shared within an organization; see the Download section for more details.

Guest     Read Only

Guests may view and interact with RCloud notebooks without having to set up an RClound account. This "view only mode" allows Data Scientists to share their completed work as a web page with non-developers. Click on an image below or visit our Gallery for additional examples.

Data Scientist     Read / Write / Execute

Since RCloud has built-in version control based on authorized Github gists, Data Scientist access is obtained through a GitHub account; click on the image below to access RCloud through an existing GitHub account or to be redirected to GitHub where you may set one up and authorize the RCloud app.

Log In

For help getting started, run RCloud Sample Notebooks which may be found in the notebook information panel that may be seen after logging into RCloud; to run a notebook, select the notebook name and then click the "run" button in the navigation bar.


Create your own notebook using the "+" key, fork notebooks using the fork icon in the navigation bar, share notebooks using the share icon (arrow in square) or begin editing by typing code in a cell; additional details are provided in the Documentation and Tutorials pages.



Download   Local Installation

Private RClouds may be created to facilitate collaboration and sharing within an organization. Local installations of RCloud have all the functionality of the public instance, including Guest and Data Scientist access permission structures. Access, sharing and security may be limited and implemented through an organization's virtual private network (VPN). Set up a private RCloud by checking out these prerequisites and local installation instructions on GitHub.
