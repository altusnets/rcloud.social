Getting Started

  • Why RCloud?

    1. Build Confidence in Data Analysis Output

      2. RCloud is an integrated computing environment (ICE) for R / Python programming which enables Data Scientists to author analyses in notebooks. Notebooks are project containers that include all the components of an analysis, including code, data, charts, technical documentation and deployment capabilities. Therefore, data analysis results may be verified simply by viewing the contents of the notebook and executing (running) the notebook. This increases user confidence in the analysis output since "black boxes" may be eliminated and results may be reproduced.

    2. Minimize Development to Deployment Effort

      3. Standard analytic workflows often involve performing a data science experiment using one set of tools and deploying the results using other tools. Rather than stitching pipelines together and/or dealing with inconsistences between development and production environments, every notebook in RCloud is named by a URL so analyses may easily be transformed from code and technical documenation to rich web-based dashboards.

    3. Increase Long-term Engagement

      4. As opposed to analysis which are stored locally, desktops or laptops for example, web-based contents allows for broad access which fosters knowledge and understanding. Data science analyses created in the RCloud environment may be accessed from anywhere as long as there is an internet connection - which could be thought of as a "social network for data science". In addition, since user access and controls remain constant, users become comfortable with navigating the environment which increases user confidence which in turn promotes user engagement.

    4. Transform Knowledge into Assets

      5. With the creation of RCloud notebooks (on the public instance or in private clouds), comes the creation of code, widget, dashboard and analysis libraries which may be reused on similar projects or to train new employees, for example. RCloud notebooks are searchable and forkable (sharable), so anyone with access to the notebooks may view and reuse a notebook or various parts of its content. Rather than spending time "recreating the wheel", RCloud users can move analyses forward by sharing relevant work within an organization or community and by adding commentary and ideas.

    5. Additional Functionality Includes Built-in Security

      6. RCloud has adopted the discipline known as Object-Capabilities (ocap) which creates a built-in security for client-server communication; read more about ocap in our Documentation or in Wikipedia. In addition, RCloud notebooks may be encrypted for added security. Additional functionality in RCloud notebooks versus other notebooks and integrated development/computing environments includes the ability to search, fork (share) and add comments to notebooks, built-in version control and multi-language support.
