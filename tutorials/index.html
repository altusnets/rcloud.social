Getting Started

  • Why RCloud?

    1. Build Confidence in Data Analysis Output

      2. Since RCloud is an integrated computing environment (ICE) for R / Python programming which enables Data Scientists to author analyses in notebooks which are containers that include all the components of an analysis, including code, data, charts, technical documentation and deployment capabilities. Therefore, data analysis results may be verified simply by viewing the contents of the notebook and executing (running) the notebook container. This increases user confidence in the analysis output since "black boxes" may be eliminated and results may be easily reproduced by re-running the notebook.

    2. Minimize Development to Deployment Effort

      3. Standard analytic workflows often involve the performance of analysis using one set of tools and deploying the results using other tools. Rather than stitching these pipelines together and/or dealing with inconsistences between development and production environments, every notebook in RCloud is named by a URL so analyses may easily be transformed from code and technical documenation to rich web-based dashboards.

    3. Increase Long-term Engagement

      4. As opposed to analysis which are stored locally, desktops or laptops for example, web-based contents allows for broad access to analyses which fosters knowledge and understanding; data science analyses created in the RCloud environment may be accessed from anywhere as long as there is an internet connection. In addition, since user access and controls remain constant, users quickly become comfortable with navigating the environment which increases engagement.

    4. Transform Knowledge into Assets

      5. As RCloud notebooks are developed (either on the public instance or in private clouds), code, widget, dashboard and analysis libraries are developed which may be reused on similar projects or to train new employees, for example. RCloud notebooks are searchable and forkable, so anyone with access to the notebooks may view and reuse content.
